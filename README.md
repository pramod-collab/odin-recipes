# odin-recipes

This project is part of The Odin Project Foundations curriculum.

The goal of this project is to build a basic recipe website using only HTML.

It helps practice structuring web pages and using common HTML elements.

 Skills Demonstrated



Writing basic HTML
 Using headings, paragraphs, lists, and links
 Adding images to a webpage
 Structuring multiple pages
Understanding how files and folders work in a project

## Project Description

The website contains a main index page that links to a few individual recipe pages.
