# Nepali Recipes Project

This is a simple HTML project that shows three popular Nepali foods: Daal Bhaat, Momo, and Selroti. The webpage includes images, short descriptions, ingredients, and cooking steps for each recipe.

## What I Learned

* Basic HTML page structure
* Adding images to a webpage
* Creating unordered and ordered lists
* Using headings and paragraphs
* Creating a simple footer

## Recipes Included

* Daal Bhaat
* Momo
* Selroti

This project was created as practice while learning HTML.
